# ShopeShortUrl
Tampermonkey 使用者指令碼，用於在 Shopee 網站的商品頁面上添加「複製短網址」按鈕，以縮短商品 URL。

## 功能說明

- 在商品頁面上自動添加「複製短網址」按鈕
- 當點擊按鈕時，會將縮短後的商品網址複製到剪貼簿
- 使用者可直接從瀏覽器擴充功能中安裝此腳本

## 安裝方式

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/) 瀏覽器擴充功能
2. 點擊 [ShopeeShortUrl.user.js](https://github.com/SakiKiaya/ShopeShortUrl/raw/main/ShopeeShortUrl.user.js) 並選擇「安裝此指令碼」

## 使用方法

1. 造訪任意 Shopee 商品頁面
2. 點擊頁面上出現的「複製短網址」按鈕
3. 網址已複製到剪貼簿，可以直接貼上使用

## 設計原理

本腳本的主要功能是從 Shopee 商品頁面中提取當前商品的短網址，并將其複製到剪貼簿。它通過以下步驟實現：
1. 檢測商品頁面載入
2. 在分享區塊添加「複製短網址」按鈕
3. 點擊時提取當前頁面 URL，移除不必要的參數，並將縮短後的 URL 複製到剪貼簿
4. 向使用者顯示成功訊息
